Normalization¶

In all regression problems it is a good idea to normalize all data. Therefore we need to create a function for normalization.

In [4]:
# Normalize each column ("attribute") of a given matrix.
# The normalization is done so that each column of the output matrix
# have 0 mean and unit standard diviation.
def normalize(x):
    return (x - np.mean(x,0)) / np.std(x,0)

Cost function¶

In most data mining problems we specify a cost function where the aim is to minimize the cost function. In linear regression the most common cost function is the mean squared error (MSE). $$ cost(Y, \hat{Y}) = \frac{1}{2m}\sum_{i = 0}^m (y_i-\hat{y_i})^2 $$

In [5]:
# The cost function
def cost(Y, Yhat):
    m = np.shape(Y)[0]
    return 1 / (2.0 * m) * np.sum((Y - Yhat) ** 2)

To be able to find the minimal cost with gradient decent we need to calculate the gradients of the cost function with respect to the weights and the bias.

$$ \frac{\partial cost}{\partial w} = \frac{1}{m}\sum_{i = 0}^m -x_i * (y_i-(x_i * w + b))$$$$ \frac{\partial cost}{\partial b} = \frac{1}{m}\sum_{i = 0}^m -1 * (y_i-(x_i * w + b)) $$
In [6]:
# The gradients of the cost function
def d_cost(Y, X, W, b):
    m = np.shape(Y)[0]
    Yhat = np.dot(X, W) + b
    dW = 1.0 / (m) * np.sum(-X * (Y - Yhat))
    db = 1.0 / (m) * np.sum(-1 * (Y - Yhat))
    return (dW, db)

The main program¶

Here we optimize the cost function using gradient decent. The cost for the train- and the test-set at each step is stored for later use.

In [9]:
energy = []
energy_test = []



for i in range(500):
    
    Y_hat = np.dot(X, W) + b
    Y_hat_test = np.dot(X_test, W) + b
    
    energy.append(cost(Y, Y_hat))
    energy_test.append(cost(Y_test, Y_hat_test))
    dW, db = d_cost(Y, X, W, b)
    W -= alpha * dW
    b -= alpha * db
